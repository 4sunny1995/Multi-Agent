# 🎨 Style Guide: Japanese Finance Login Screen

## 1. Thiết kế ý tưởng (Concept)
- **Tông màu**: Sự kết hợp giữa Navy (#1A365D) và Trắng sứ để tạo cảm giác **Tin cậy** và **Chuyên nghiệp**.
- **Phong cách**: Minimalism (Tối giản), Tôn trọng "Ma" (Khoảng trống).
- **Ngôn ngữ**: Đa ngôn ngữ (VI/JP).

## 2. Bảng màu (Color Palette)

| Loại | Hex Code | Tên màu | Ứng dụng |
| :--- | :--- | :--- | :--- |
| **Primary** | `#1A365D` | Trust Navy | Header, Buttons, Links |
| **Secondary** | `#E2E8F0` | Soft Gray | Borders, Placeholder |
| **Background** | `#FFFFFF` | Porcelain | Main Canvas |
| **Success** | `#38A169` | Secure Green | Success notifications |
| **Error** | `#E53E3E` | Alert Red | Input errors |

## 3. Typography (Chữ viết)
- **Primary**: `Noto Sans JP` (Gothic) - Phông chữ tiêu chuẩn cho giao diện kỹ thuật số Nhật Bản.
- **Weights**: 
    - Bold (700) : Cho tiêu đề (Login/ログイン).
    - Regular (400): Cho mô tả và nội dung input.

## 4. UI Elements (Thành phần giao diện)
- **Buttons**: Bo góc nhẹ (4px), Shadow mảnh tạo chiều sâu.
- **Inputs**: Đường viền 1px màu Soft Gray, Focus state chuyển sang Trust Navy.
